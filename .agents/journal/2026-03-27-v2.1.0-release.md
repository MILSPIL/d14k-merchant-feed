## 2026-03-27: Реліз v2.1.0 (SaaS UI, Custom Labels, Розклади)

**Що було зроблено:**

- **SaaS Редизайн:** Повністю перероблено інтерфейс налаштувань. Впроваджено темну шапку, таб-навігацію, карткову структуру та дизайн-систему на базі CSS-змінних (стандартизовані відступи 16px).
- **Виправлення UI:** Виправлено баг з надмірним padding у дереві категорій через проблеми CSS-специфічності. Виправлено SVG іконку довідки (встановлено видимий радіус крапки `1.5px`).
- **Створена вкладка "Довідка":** Додано інструкції та контактний банер розробника (D14K).
- **Custom Labels (0-4):** Розібрано логіку для Google Ads. Тепер можна призначати мітки масово через маппінг категорій і перекривати їх індивідуально через кастомні мета-поля товару.
- **Розширена фільтрація:** Додано можливість виключати товари за категоріями (автоматично враховує всі WPML-переклади категорії), атрибутами та через кастомні фільтри.
- **Нові теги XML:** Додано `g:country_of_origin` та `g:shipping`. Бренд тепер можна тягти з будь-якого атрибута.
- **Оптимізація розкладу (CRON):** Видалено неактуальні інтервали оновлення (3, 6, 12 годин), оскільки Google Merchant Center забирає фід мінімум раз на добу. Додано можливість задавати точний час генерації фіду з урахуванням локального часового поясу.
- **Документація:** Оновлено `README.md` (чистий changelog без AI-маркерів).
- **Деплой:** Реліз зафіксовано на GitHub (`v2.1.0`) і розгорнуто на трьох сайтах (14karat, techmachagro, filler).
- **Фікс UI:** Винесено блок з відмовою від відповідальності окремо від загальної картки, налаштовано відступ 20px та внутрішні padding 16px.
